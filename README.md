zeShop
===================
ZeShop is a free, open source e-commerce solution. 
The solution is build with micro-sevices architecture based on spring-cloud framework.

----------


Getting Started (DEV)
-------------
###Server order :

- Configuration server
- Eureka server
- Auth server
- Product server
- Cart server
